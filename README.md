## Требования
* [Node.js](https://nodejs.org)

## Локальный запуск
1. Установка зависимостей
```bash
npm i
```
2. Запуск среды разработки
```bash
nmp run dev
```
По умолчанию сервер слушает [localhost:3000](http://localhost:3000)

## Деплой
Для компиляции стилей используется скрипт:
```bash
npm run build-scss
```
